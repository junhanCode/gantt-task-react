## 🚀 启动 Demo

### 方法1: 快速启动（推荐）

```bash
# 1. 进入项目根目录
cd d:\code\gantee\gantt-task-react

# 2. 构建库（如果还没构建）
npm run build

# 3. 进入example目录
cd example

# 4. 安装依赖（首次运行）
npm install

# 5. 启动demo
npm start
```

访问: `http://localhost:3000`

### 方法2: 开发模式

如果需要同时修改库和demo：

**终端1 - 监听库变化**:
```bash

```

**终端2 - 运行demo**:
